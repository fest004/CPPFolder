Logging for logging events
A way  for the engine telling us what happens, game creator or user
Abundance of information can be logged:
    Can log system information
    Whether files have been opened
    Wheter this ever happened or not
    Warnings and errors
    When things go wrong, it is important to pinpoint where, when and how




We are using spdlog framework, but it is important to wrap major features that come from libraries, so if we change
the lib in the future, we edit the contents inside the wrapper to not break things client side
